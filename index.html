Help & Documentation

Getting Started

The Zero Trust Total Cost Analyzer helps you compare the total cost of ownership (TCO) of Portnox Cloud with other NAC solutions.

How to Use

  1. Configure your analysis parameters in the sidebar
  2. Select vendors to compare with Portnox Cloud
  3. Click "Calculate TCO" to generate the comparison
  4. View different perspectives using the tabs above

Understanding the Results

The TCO analysis includes all direct and indirect costs associated with each solution over the selected time period, including:

  • Hardware and software costs
  • Implementation and professional services
  • Ongoing maintenance and support
  • Personnel costs (FTE requirements)
  • Risk-adjusted costs

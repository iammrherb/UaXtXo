Getting Started

This tool helps you calculate and compare the Total Cost of Ownership (TCO) of different Network Access Control (NAC) solutions.

How to Use

  1. Select Your Current Vendor - Choose your current NAC solution or "No NAC" if you don't have one.
  2. Select Your Industry - This provides relevant compliance frameworks and benchmarks.
  3. Enter Organization Details - Customize the analysis to your environment.
  4. Configure Costs - Fine-tune pricing parameters for accurate comparison.
  5. View Results - Analyze detailed comparison, ROI, and risk analysis.

Understanding the Results

The results are organized in several tabs:

  • Comparison - High-level TCO comparison between your current solution and Portnox Cloud.
  • Cost Details - Breakdown of costs by category.
  • Implementation - Timeline comparison and migration strategy.
  • Features - Feature comparison between solutions.
  • ROI - Return on investment analysis.
  • Risk Analysis - Security risk and compliance impact.
  • Sensitivity - How changes in parameters affect TCO.

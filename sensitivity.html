Sensitivity Parameters

Configure the parameters for sensitivity analysis.

Analyze how changes in the total number of devices affect TCO and relative savings. More devices typically increase hardware and licensing costs for on-premises solutions.

How to Use Sensitivity Analysis

  1. Select the variable you want to analyze
  2. Choose which vendor(s) to include in the analysis
  3. Specify the range of values to test
  4. Run the analysis to see how the variable affects TCO
  5. Review charts, tables and breakeven analysis
  6. Save scenarios to compare different analyses

Sensitivity analysis helps identify how specific factors impact the total cost of ownership. This is valuable for making informed decisions based on your organization's specific priorities and risk factors.
